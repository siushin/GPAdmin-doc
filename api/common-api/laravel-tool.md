# LaravelTool

LaravelTool 是一个 Laravel 扩展包，提供了常用的工具类和特征。

**包位置**：`siushin/laravel-tool`

## Traits

### ModelTool

**位置**：`Siushin\LaravelTool\Traits\ModelTool`

**方法：**

- `fastGetPageData()` - 快速获取分页数据
- `buildWhereData()` - 构建查询条件数组

## Utils

### Tree

**位置**：`Siushin\LaravelTool\Utils\Tree`

**方法：**

- `getTree()` - 将扁平数据转换为树形结构

## Cases

### Json

**位置**：`Siushin\LaravelTool\Cases\Json`

**作用：** JSON 类型转换类，用于 Eloquent 模型的 casts

## Enums

### RequestSourceEnum

**位置**：`Siushin\LaravelTool\Enums\RequestSourceEnum`

**作用：** 请求来源枚举（PC端、移动端等）

### SocialTypeEnum

**位置**：`Siushin\LaravelTool\Enums\SocialTypeEnum`

**作用：** 社交账号类型枚举

### GenderTypeEnum

**位置**：`Siushin\LaravelTool\Enums\GenderTypeEnum`

**作用：** 性别类型枚举

### UploadFileTypeEnum

**位置**：`Siushin\LaravelTool\Enums\UploadFileTypeEnum`

**作用：** 上传文件类型枚举
