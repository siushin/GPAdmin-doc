# Base 模块

Base 模块提供了系统的基础功能，包括用户认证、权限管理等核心功能。

## 功能特性

- 用户认证与授权
- 权限管理
- 基础数据管理
- 命令行工具

## 快速开始

Base 模块是 GPAdmin 的核心模块，提供了系统运行所需的基础功能。
