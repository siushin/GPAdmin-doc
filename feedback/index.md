# 意见反馈

感谢您使用 GPAdmin！我们非常重视您的反馈和建议。

## 反馈渠道

### GitHub Issues

如果您发现了 Bug 或有功能建议，请在 GitHub 上提交 Issue：

🔗 [提交 Issue](https://github.com/siushin/GPAdmin/issues)

### 反馈内容

提交反馈时，请包含以下信息：

1. **问题描述**: 详细描述遇到的问题或建议
2. **复现步骤**: 如果是 Bug，请提供复现步骤
3. **预期行为**: 描述您期望的行为
4. **实际行为**: 描述实际发生的行为
5. **环境信息**:
   - 操作系统
   - PHP 版本（后端问题）
   - Node.js 版本（前端问题）
   - 浏览器版本（前端问题）

### Issue 模板

提交 Issue 时，请使用以下格式：

```markdown
## 问题描述
简要描述问题

## 复现步骤
1. 
2. 
3. 

## 预期行为
描述期望的行为

## 实际行为
描述实际发生的行为

## 环境信息
- 操作系统: 
- PHP 版本: 
- Node.js 版本: 
- 浏览器版本: 
```
