# 模块开发

GPAdmin 前端采用模块化架构，每个模块都是独立的功能单元。

## 模块结构

每个模块都遵循统一的目录结构，便于开发和维护。

## 创建新模块

### 使用命令行工具

```bash
npm run create:module ModuleName
```

### 手动创建

1. 在 `src/pages` 目录下创建模块文件夹
2. 按照标准目录结构创建必要的文件和目录
3. 配置路由和菜单

## 模块配置

每个模块都需要在路由和菜单配置中注册：

- 路由配置
- 菜单配置
- 权限配置
