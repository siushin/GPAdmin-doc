# 快速开始

## 环境要求

- Node.js >= 14.x
- npm >= 6.x 或 yarn >= 1.x

## 安装步骤

1. 克隆项目

```bash
git clone https://github.com/siushin/GPAdmin.git
cd GPAdmin/GPAdmin
```

2. 安装依赖

```bash
npm install
```

3. 启动开发服务器

```bash
npm start
```

4. 访问应用

打开浏览器访问 `http://localhost:8000`
